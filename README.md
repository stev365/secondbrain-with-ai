# secondbrain-with-ai
description: Concise instruction for you to build an digital second brain with obsidian and ai (claude,gemini cli, etc..)

Instructions:

You will need Obsidian and an ai agent like Claude code or Gemini CLI installed in your computer. Create an obsidian vault in directory of your choice and open the ai agent cli in that directory. After that, you simply need to feed OBSIDIAN-SECOND-BRAIN-GUIDE.md to the agent and the ai will do the rest for you. Enjoy the second brain! :)
